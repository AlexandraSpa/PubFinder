2. Web app structure

2.1. Home Page

The home page offers the users the possibility to search any pub location without being logged in and also the possibility to register.

2.2. Navigation menu

The navigation menu offers the possibility of finding the closer pub pin using the user location, also offers the options of register or log in and finally, it can redirect the users to the social media accounts of the app.

2.3. User Profile login and register

3. QOC

  • How can the user save a pub's location? The user must first create a free account. After that, all he has to do is click the option from the top menu: Save your favourite pubs.
  • How can a pet owner find a pub that didn't have restrictions concerning pets? The user can access the filter option: Pet friendly.
  • How can a pub owner register their pub into the app? For new pubs to appear on the map, the owners of the pubs can register their locations and enjoy the many clients that will come.

4. Personas

  • Pet owner: A pet owner needs to find a pub that didn't have pets restrictions. He can choose the special filter: Pet friendly that will redirect him to this specific type of pub.
  • Pub owner: A new pub owner can register the bussiness to the app for free, so that he can promote the pub and attract new customers. He also have the possibility to colaborate with the PubFinder team for special sales.
  • Student owner: A student owner is mostly new to town so he/she can use the app to easily find new pubs everywhere he/she goes.
